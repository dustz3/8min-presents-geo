## 使用方法

### 基本操作

1. **開啟簡報**

   - 在瀏覽器中開啟 `index.html` 文件

2. **導航控制**

   - **按鈕控制**：使用畫面底部的左右箭頭按鈕
   - **鍵盤控制**：
     - `→` 或 `空白鍵` 或 `Enter`：下一張
     - `←`：上一張
     - `Home`：第一張
     - `End`：最後一張
     - `F`：切換全螢幕

3. **觸控支援**
   - 在移動設備上可以左右滑動切換投影片

### 進階功能

- **自動播放**：可在控制台執行 `presentationControls.startAutoPlay()` 啟動
- **停止自動播放**：執行 `presentationControls.stopAutoPlay()`
- **全螢幕模式**：按 `F` 鍵或執行 `presentationControls.toggleFullscreen()`
